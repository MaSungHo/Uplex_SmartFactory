# Uplex_SmartFactory

<h3> 유플렉스 스마트공장 API 서비스</h3>

django version: 3.1.2 <br>
python version: 3.8.3 <br><br>


<h3> - 실행 명령어 </h3>
python manage.py runserver


<h3> - 이미지 모듈 </h3>
Django의 ImageField를 사용하기 위해 Pillow 설치가 필요 <br>
- <b>pip install Pillow</b> 

<h3> - InfluxDB </h3>
현재 로컬에서 InfluxDB를 설치하여 사용
